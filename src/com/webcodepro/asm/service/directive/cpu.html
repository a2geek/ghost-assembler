.cpu <name>

Load CPU definition from the predefined list.

Multiple .cpu directives may be used in the source file to switch between CPU definitions (for instance, on the Apple II series to switch between 6502 and Sweet-16).

Example: 

    .cpu <MOS6502>
    lda #5
    sta 0x200
 
    .cpu <SWEET16>
    ld @R1
    st @R2

.cpu "name"

Load custom CPU from the file system. Relative or absolute path names are allowed.

Example: 

     .cpu "my_68000.xml" 
     moveq #0, d0
     moveq #0, d1
loop move.w (a0)+, d0
     add.l d0, d1
     dbra d2, loop

Usage

Assembler is a Java-based application that requires Java 6 installed on your system. This application is command-line driven at the moment and only command-line. Therefore, invocation is a little bit different than a native command-line application, and is invoked in the following manner:

java -jar Assembler-x.y-yyyymmdd.hhmm.jar <options> files...

Note that this application uses a utility called One-JAR to package all dependencies into a single JAR file, removing all CLASSPATH setup.

The following command-line options are available:

${pd.names}
${pd.description}
